1-Mise en place (Pour Raspberry Pi Zero 2W)

	Flash BookwormLite  (https://downloads.raspberrypi.com/raspios_oldstable_lite_armhf/images/raspios_oldstable_lite_armhf-2025-11-24/2025-11-24-raspios-bookworm-armhf-lite.img.xz)
	
	Sur RPImager mise en place de la configuration, name : rpi password : rpi
	Toujours sur le RPImager mettre en place le wifi sur lequel elle devra ce connecter.
	
	Dépot des doc et script dans /home/rpi (sur la SD)
	
	Ejecter la carte via le PC (Pour éviter les prblm ensuite)
	
	Mise en place de carte, mise en tension de la Raspberry
	Connection avec rpi/rpi
	
	Lancement du script.sh avec la commande sudo bash Script.sh 2
	
	Laissé l'instalation ce faire puis enjoy.
